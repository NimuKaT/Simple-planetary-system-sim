How to create objects.

To create an object you go the Object Creating window.
From there you can customise your own object giving the colour of the object, the radius (in kilometers) and the material (aka. the density).
Or generate a random object, using the button provided. Once you have done one of these options, move your mouse over the space box. Click the mouse to place the object in the selected place.
Once the object has been placed click again to set the velocity of the new object. This object has now been created. You can create as many objects as you would like and watch them interact with each other.
